# React + Vite


## Installation

### Prerequisites

- **Node.js** and **npm** (or **Yarn**)
- **Git**
- A working **Laravel** backend (or other compatible backend service)

### Clone the Repository

```bash
git clone https://github.com/your-username/whois-frontend.git
cd whois-frontend

Install Dependencies
npm install
npm run dev
